# Claude Code Skills - エージェントスキルガイド

このスキルは、Claude Code のエージェントスキル機能に関する包括的な知識を提供します。

## 概要

Claude Code のスキル機能を理解し、効果的に作成・運用するための完全ガイドです。スキルの基本概念から、ファイル構造、YAML 設定、ベストプラクティスまでをカバーしています。

## このスキルを使用する場面

- スキルを新規作成したいとき
- SKILL.md ファイルの書き方を知りたいとき
- YAML メタデータの設定方法を確認したいとき
- スキルがトリガーされない問題をトラブルシューティングするとき
- ツールアクセス制限の方法を知りたいとき
- スキルの配布方法を検討しているとき

## トリガーキーワード

このスキルは以下のようなキーワードでトリガーされます：

- 「スキルを作成」
- 「SKILL.md の書き方」
- 「エージェントスキル」
- 「skill」「skills」
- 「allowed-tools」
- 「user-invocable」
- 「スキルのトラブルシューティング」

## 主な機能

### 1. スキルの基本概念
- スキルとは何か
- モデル呼び出し型の動作
- 3段階のプロセス（Discovery、Activation、Execution）

### 2. ファイル構造
- 必須の SKILL.md ファイル
- オプションのサポートファイル
- 段階的開示の原則

### 3. YAML メタデータ
- 必須フィールドとオプションフィールド
- ツールアクセス制限
- コンテキストフォーク設定

### 4. ベストプラクティス
- 効果的な説明の書き方
- 複数ファイル構造の設計
- トリガーキーワードの選定

### 5. トラブルシューティング
- スキルがトリガーされない問題
- スキルがロードされない問題
- 複数スキルの競合解決

## 使用例

### 基本的な使い方

```
ユーザー: 新しいスキルを作成したいです
Claude: [このスキルがトリガーされ、スキル作成の手順を提供]
```

### トラブルシューティング

```
ユーザー: 作成したスキルがトリガーされません
Claude: [このスキルがトリガーされ、トラブルシューティング手順を提供]
```

## ファイル構成

```
claude-code-skills/
├── SKILL.md           # メインスキルファイル（包括的なガイド）
├── metadata.json      # スキルのメタデータ
└── README.md          # このファイル（概要と使い方）
```
